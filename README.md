# VoiceLab Studio

VoiceLab Studio — Telegram-бот для AI-преобразования голоса, дубляжа видео, генерации субтитров и перевода озвучки.

Проект является современной переработкой идеи `RVC-telegram-bot`: вместо старого монолитного решения используется архитектура с API, очередью задач, worker-процессами и Docker-деплоем.

## Возможности

Планируемые возможности проекта:

- преобразование голоса через RVC;
- обработка голосовых сообщений, аудио и видео;
- генерация AI-каверов;
- извлечение аудио из видео;
- сборка результата обратно в видео;
- транскрибация аудио/видео;
- перевод речи через Gemini API;
- генерация субтитров `.srt` / `.ass`;
- дубляж видео на другой язык;
- очередь задач с отображением прогресса;
- поддержка GPU через Docker.

## Основная идея

Бот не выполняет тяжёлую обработку сам. Он только принимает файлы, создаёт задачи, показывает прогресс и отправляет результат пользователю.

Тяжёлая обработка выполняется отдельными worker-процессами.

```text
Telegram user
    ↓
Telegram bot
    ↓
API service
    ↓
Redis queue
    ↓
Workers:
    - rvc-worker
    - media-worker
    - gemini-worker
    ↓
Storage
```

## Docker и GPU

GPU требуется только для worker-процесса, который выполняет RVC-инференс и другие локальные нейросетевые операции.

API, Telegram-бот, Redis и база данных не должны требовать видеокарту.

Примерная схема сервисов:

```text
bot           — Telegram-интерфейс
api           — HTTP API и управление задачами
redis         — очередь задач
db            — база данных
rvc-worker    — обработка RVC с GPU
media-worker  — ffmpeg-операции
gemini-worker — транскрибация/перевод через Gemini API
```

## MVP

Первый полезный MVP:

```text
Пользователь отправляет voice/audio
→ бот создаёт задачу
→ worker преобразует голос через RVC
→ бот отправляет готовый audio-файл
```

После стабильного MVP можно добавлять перевод видео, субтитры и дубляж.
